How To Install and Use OpenFOAM (open source CFD) for the Absolute Beginner

All code can be found at: https://github.com/lordvon/OpenFOAM_Tutorials/tree/master/Beginner

DESCRIPTION:
Here we will go over how to install OpenFOAM on Ubuntu Linux, and run a tutorial case. The flow field solution will be viewed with ParaView.

OUTLINE:
-Follow Instructions on OpenFOAM website.
-Add desired settings.
-Copy a tutorial case from $FOAM_TUTORIALS.
-Explain the file directories.
-Run a case and view results.

This tutorial was run successfully on:
-Ubuntu 14.04 64-bit
-OpenFOAM 2.4.0
